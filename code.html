                            function checkForPalindrone(initValue) {
                            
                                // convert to lowercase
                                initValue = initValue.toLowerCase();
                                //remove any spaces and other special characters
                                let regex = /[^a-z0-9]/gi;
                                initValue = initValue.replace(regex, "");
                            
                                let revString = [];
                                let returnObj = {};
                            
                                for (let index = initValue.length - 1; index >= 0; index--) {
                                    revString += initValue[index];
                                }
                            
                                if (revString == initValue){
                                    returnObj.msg = "Well Done! You've entered a Palindrone.";
                                }
                                else {
                                    returnObj.msg = "Sorry, you did not enter a Palindrone.";
                                }
                            
                                returnObj.reversed = revString;
                                return returnObj;
                            }

                            function displayMessage(returnObj) {
                                document.getElementById("alertHeader").innerHTML = returnObj.msg;
                                document.getElementById("msg").innerHTML = `Your phrase reversed is ${returnObj.reversed}`;
                                // make the alert visible
                                document.getElementById("alert").classList.remove("invisible");
                            }
ROTATOR

Palindrone check (aka ROTATOR) is an app that takes a user's input and checks to see if it is a Palindrone.

The code is structured in three functions.

The first function getString() receives the input from the form and calls the checkForPalindrone() and displayMessage() functions.

The next function checkForPalindrone() creates a new array by using a for/loop in reverse. It then compares the reversed string and the original input string.

The last function displayMessage() displays the corresponding message result on the page based on whether or not the string was a palindrone.
